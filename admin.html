⚠️ Backend Setup Required

To make this dashboard work, create a Supabase project and add your credentials to config.js.
Also, run the following SQL in your Supabase SQL Editor:

CREATE TABLE menu_items (
  id uuid DEFAULT uuid_generate_v4() PRIMARY KEY,
  name TEXT, description TEXT, price TEXT,
  category TEXT, emoji TEXT, is_best_seller BOOLEAN
);
CREATE TABLE inquiries (
  id uuid DEFAULT uuid_generate_v4() PRIMARY KEY,
  customer_info TEXT, item_name TEXT, created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
